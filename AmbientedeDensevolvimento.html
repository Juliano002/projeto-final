Como desenvolvemos este site

Para desenvolvermos este site,utilizamos HTML5 e CSS3 pelo Visual Studio.

Dividimos as páginas em tags do tipo "header, nav, main, ul, footer..." para facilitar a estilização.

Utilizamos o GitHub para compartilharmos o código e desenvolve-lo ao mesmo tempo em computadores diferentes.
